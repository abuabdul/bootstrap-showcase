JavaScript Plugins

Bootstrap provides many Javascript plugins. Plugins can be included in your application individually or all at once. Bootstrap gives you the flexibility to use its markup API, first class API of Bootstrap, that uses plugins without writing any line of Javascript code. Each plugin can be used with this markup API. Also you can use Javascript API for these plugins. Examples of each plugin and its usage are documented in Boostrap site.

For more details on JavaScript plugins, visit Bootstrap Javascript
