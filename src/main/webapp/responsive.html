Responsive Bootstrap

Bootstrap has an amazing feature of responsiveness to the devices. It knows how to respond to the device in which Bootstrap application is viewed. Bootstrap responsive design helps us to complement the same site to target different devices like desktop, tablet, phone etc., instead of creating different sites for each target device. Using Bootstrap responsive design, mobile friendly applications can be built without much hassle.
